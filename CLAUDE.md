# gstack

Use the `/browse` skill from gstack for all web browsing. Never use `mcp__claude-in-chrome__*` tools directly.

Available gstack skills:
- `/office-hours` — async design/eng feedback session
- `/plan-ceo-review` — CEO review of a plan
- `/plan-eng-review` — engineering review of a plan
- `/plan-design-review` — design review of a plan
- `/design-consultation` — design consultation
- `/design-shotgun` — rapid design exploration
- `/design-html` — generate HTML design mockup
- `/review` — code review
- `/ship` — ship a feature end-to-end
- `/land-and-deploy` — land and deploy a change
- `/canary` — canary deploy
- `/benchmark` — benchmark performance
- `/browse` — web browsing (use this for ALL web browsing)
- `/connect-chrome` — connect a Chrome browser
- `/qa` — QA a feature
- `/qa-only` — QA without implementation
- `/design-review` — review designs
- `/setup-browser-cookies` — set up browser cookies
- `/setup-deploy` — set up deployment
- `/setup-gbrain` — set up gbrain
- `/retro` — run a retrospective
- `/investigate` — investigate an issue
- `/document-release` — document a release
- `/document-generate` — generate documentation
- `/codex` — run codex
- `/cso` — chief of staff operations
- `/autoplan` — auto-generate a plan
- `/plan-devex-review` — devex review of a plan
- `/devex-review` — developer experience review
- `/careful` — careful/cautious mode
- `/freeze` — freeze the codebase
- `/guard` — guard mode
- `/unfreeze` — unfreeze the codebase
- `/gstack-upgrade` — upgrade gstack
- `/learn` — learn about a topic
